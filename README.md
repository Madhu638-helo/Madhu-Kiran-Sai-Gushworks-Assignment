# Meeara HDPE Pipes Product Landing Page

A responsive, single-page product website for **Meeara HDPE Pipes**, built with plain HTML, CSS, and JavaScript.

## Overview

This project showcases a modern product landing experience with interactive UI components, including:

- Sticky header that appears on scroll
- Product image carousel with thumbnail navigation
- Hover zoom preview for product images
- Mobile hamburger menu
- Modal popups (datasheet download and quote request)
- Smooth-scroll anchor navigation
- Tabbed manufacturing process section
- Industries carousel and additional interactive sections

## Tech Stack

- HTML5 (`index.html`)
- CSS3 (`styles.css`)
- Vanilla JavaScript (`script.js`)

No frameworks or build tools are required.

## Project Structure

- `index.html`: Page structure and semantic sections
- `styles.css`: Full visual styling, responsive behavior, and animations
- `script.js`: All interactive behaviors and event handling

## Run Locally

Because this is a static website, you can run it in either of these ways:

1. Open `index.html` directly in your browser.
2. Or use a local server (recommended):
   - With VS Code Live Server extension, click **Go Live**.
   - Or run a simple server from this folder:

```bash
python3 -m http.server 5500
```

Then open `http://localhost:5500` in your browser.

## Customization Guide

- Update content text in `index.html`.
- Replace product images/URLs in the carousel section of `index.html`.
- Adjust brand colors and spacing variables in `styles.css` under `:root`.
- Modify interaction behavior (carousel, sticky bar, modals, menu) in `script.js`.
